# Registration & Login

## Creating an Account

1. Navigate to `/register`
2. Fill in the registration form:

| Field | Description |
|-------|-------------|
| **Username** | Your login name |
| **Email** | Your email address |
| **Password** | Choose a strong password |
| **Display Name** | Your public-facing name |

3. Click **Register**
4. You are automatically logged in

## Logging In

1. Navigate to `/login`
2. Enter your username and password
3. Click **Login**
4. You are redirected to the admin dashboard

## Logging Out

Click **Logout** in the admin navigation menu or navigate to `/logout`.

## Password Recovery

Password recovery is not yet implemented. Contact an administrator to reset your password.
